# Сценарии тестирования — Timeline

Папка содержит сценарии ручного (интеграционного) тестирования ключевых функций приложения.

## Формат сценария

Каждый сценарий — это один `.md`-файл со следующей структурой:

| Раздел | Содержание |
|--------|-----------|
| **Цель** | Что проверяем |
| **Preconditions** | Что должно быть готово перед прогоном |
| **Шаги** | Пошаговая инструкция |
| **Протокол** | Таблица с датой/временем, командой, результатом каждого шага |
| **Вывод** | Pass / Fail, примечания |

## Запуск

Сценарии рассчитаны на дев-окружение: `localhost:3001` (API), сидовые данные (`admin`/`admin`).

Артефакты (экспортированные `.xlsx` и т.п.) складываются в `localtemp/`.
